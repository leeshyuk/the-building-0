## 게임 흐름
1층에서 4층 강의실에 두고 온 물건을 찾으러 엘리베이터를 탐.   
엘리베이터가 갑자기 오작동을 일으킨 후 문이 열리는 데 1층의 구조가 4층처럼 변해있음.   
4층까지 8번 출구처럼 플레이 - 나중에 쓰일 비밀번호 단서도 등장   
4층 도달 시 물건 찾기   
내려가려는 데 귀신이 추격   
1층에 갔는데 나가는 문이 잠겨있음   
비밀번호를 입력해서 탈출   

이동윤
> 초반엔 액자의 그림이 변경되는 등(ex.액자의 초상화에서 갑자기 눈물 흘림)같은 단순한 요소를 제공하다가 후반부로 갈수록 갑툭튀 요소를 많이 추가해서 공포를 극대화
> 뒤돌아볼 수 있는 미끼 요소를 추가해서 뒤돌았을 때 갑자기 귀신이 나타나는 등 무서운 효과도 생각해봤음

6. 멀리서 사람의 형태가 희미하게 보이다가 사라지는 현상
